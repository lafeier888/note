# 疑问

为什么用分布式?

答:这个问题要说到不同工具的对比

svn:集中式,服务器挂了,都不能干活

git是分布式的,就是说有本地仓库和远程仓库,计算没联网,远程仓库挂了,依然可以干活,网络恢复在同步一下仓库就行
